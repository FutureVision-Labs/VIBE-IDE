üöÄ Getting Started

Welcome to VIBE IDE!

VIBE IDE is a beginner-friendly coding environment designed to make programming accessible, fun, and collaborative. Whether you're a complete beginner or an experienced developer, VIBE IDE has something for everyone.

VIBE IDE is available as a desktop application. Download it from:

  • itch.io: futurevisionlabs.itch.io/vibe-ide
  • GitHub: github.com/FutureVision-Labs/VIBE-IDE

  2. Go to File ‚Üí New Project
  3. Choose a project template:
    • üìÑ Blank Project - Start from scratch with a clean slate

    • üêç Python Beginner - Simple Python script to learn the basics

ü§ñ Cursy's Corner

Meet Cursy

Cursy is your AI coding assistant, powered by GPT-4o-mini. Cursy doesn't just wait for questions‚Äîshe observes, suggests, and helps you learn as you code!

Chatting with Cursy

To chat with Cursy:

  1. Open Cursy's Corner (usually in the right sidebar)
  2. Type your question or request in the chat input
  3. Press Enter or click Send
  4. Cursy will respond with helpful information, code examples, or suggestions

What Cursy Can Do

  • Answer coding questions
  • Explain code concepts
  • Help debug errors
  • Suggest code improvements
  • Generate code snippets
  • Update project journals
  • Create CML files
  • And much more!

Customizing Cursy

You can customize Cursy's personality through File ‚Üí Agent Persona Settings:

  • Choose from pre-made templates (Friendly Helper, Fun Friend, Strict Teacher, etc.)
  • Set personality traits (friendly, funny, patient, etc.)
  • Adjust communication style (conversational, professional, playful, etc.)
  • Control emoji usage
  • Add special instructions
üí° Tip: You can set a global persona that applies to all projects, or allow project-specific personas to override it!

Cursy's Notes

Cursy's Corner includes a notes feature where you can:

  • Jot down quick ideas
  • Save code snippets
  • Keep reminders
  • Store project-related notes

Notes are saved locally and always accessible while coding. Click the corkboard or pinned notes in Cursy's office to open the notes editor.

üíª Code Editor

Monaco Editor

VIBE IDE uses the Monaco Editor‚Äîthe same powerful editor that powers VS Code and Cursor. This means you get:

  • Intelligent autocompletion
  • Syntax highlighting for many languages
  • Code folding
  • Multi-cursor editing
  • Find and replace
  • Bracket matching
  • And much more!

Opening Files

To open a file:

  1. Use File ‚Üí Open File from the menu
  2. Or click on a file in the file tree
  3. The file will open in the editor

Saving Files

To save a file:

  • Press Ctrl+S (Windows/Linux) or Cmd+S (Mac)
  • Or use File ‚Üí Save from the menu
  • Use File ‚Üí Save As to save with a new name

Editor Features

  • Syntax Highlighting: Automatic color coding based on file type
  • Autocompletion: Press Ctrl+Space for suggestions
  • Go to Line: Press Ctrl+G to jump to a specific line
  • Find: Press Ctrl+F to search in the current file
  • Replace: Press Ctrl+H to find and replace
  • Multi-cursor: Hold Alt and click to add multiple cursors

üìù CML System

What is CML?

CML (Compressed Markup Language) is a custom format for tracking events, decisions, and progress in your projects. CML files can be rendered into narrative stories, devlogs, or timelines.

CML Viewer

To view and edit CML files:

  1. Go to File ‚Üí CML Viewer
  2. Click Open CML File to load an existing file
  3. Or click New CML File to create one
  4. Edit the CML source in the left panel
  5. Click Render to Story to see the narrative version
  6. Click Save CML to save your changes

Chat-to-CML Recording

You can automatically record chat conversations to CML:

  1. Open File ‚Üí CML Viewer
  2. Check the Record Chat to CML checkbox
  3. All future conversations with Cursy will be saved as CML files in your project's cml/ folder
üí° Tip: CML files are saved with timestamps, so you can track your project's evolution over time!

CML Format

CML uses a structured format:

[timestamp|type|participants|location|tags]{
  title:"Event Title";
  description:"Event description";
  // ... more fields
}

For more information about CML syntax, check the CML Viewer help section.

  • Browse results and click play to preview

Downloading Audio

üîß Troubleshooting

Common Issues

Cursy Not Responding

  • Check if OpenAI API key is configured in settings
  • Verify your internet connection
  • Try reloading the application
  • Check the console for error messages

Files Not Saving

  • Make sure you have write permissions for the project folder
  • Check if the file is read-only
  • Try saving with a different name (Save As)
  • Check the console for error messages

Audio Not Playing

  • Check your system volume
  • Try a different audio track
  • Check your internet connection (for streaming)
  • Reload the Music & SFX panel

Code Not Implementing

  • Make sure you have a project open
  • Check that the code blocks are properly formatted
  • Verify file paths in the code blocks
  • Check the console for error messages

Getting Help

‚ùì Frequently Asked Questions

What is VIBE IDE?

VIBE IDE is a beginner-friendly coding environment with AI assistance. It's designed to make programming accessible, fun, and collaborative for beginners, kids, and experienced developers alike.

Is VIBE IDE free?

Yes! The BYO (Bring Your Own) key version is completely free! You can use VIBE IDE for free by providing your own OpenAI API key. VIBE IDE is also available on itch.io with additional options. Check the itch.io page for current pricing and availability.

What programming languages does VIBE IDE support?

VIBE IDE supports all programming languages through the Monaco Editor. It includes syntax highlighting, autocompletion, and other features for popular languages like JavaScript, Python, HTML, CSS, and many more.

How do I configure the OpenAI API key?

To use Cursy, you'll need to configure your OpenAI API key. The configuration file location is shown in the console when you start VIBE IDE. Create a JSON file with your API key: {"apiKey": "your-key-here"}

Can I use VIBE IDE offline?

Most features work offline, but Cursy (the AI assistant) requires an internet connection and OpenAI API access. File editing, project management, and other core features work completely offline.

What is CML?

CML (Compressed Markup Language) is a custom format for tracking events, decisions, and progress in your projects. CML files can be rendered into narrative stories, devlogs, or timelines. It's a unique feature of VIBE IDE that helps you document your coding journey.

How do I update my project journal?

You can ask Cursy to update your project journal by saying "update journal" or "cursy, update the journal" in the chat. Cursy will analyze your project and automatically update the PROJECT_JOURNAL.md file with current information.

Can I customize Cursy's personality?

Yes! Go to File ‚Üí Agent Persona Settings to customize Cursy's personality. You can choose from templates, set personality traits, adjust communication style, control emoji usage, and add special instructions.

Where are my projects saved?

Projects are saved in the location you choose when creating them. By default, projects are saved in your Documents folder, but you can choose any location you prefer.

How do I report a bug or request a feature?

You can report bugs or request features through the GitHub repository or contact support through the itch.io page.
