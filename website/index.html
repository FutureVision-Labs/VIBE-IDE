The Story

VIBE IDE started as GameForge - a Phaser.js-exclusive game development environment. But we realized the core IDE infrastructure could be so much more.

We evolved it into VIBE IDE - a beginner-friendly coding environment that makes programming accessible to everyone. No gatekeeping. No intimidation. Just collaboration, learning, and building.

Built with transparency, designed for beginners, and powered by the "vibe coding" philosophy - where AI assistance is collaboration, not cheating.
