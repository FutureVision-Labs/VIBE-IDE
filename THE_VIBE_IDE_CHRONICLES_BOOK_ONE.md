### The Solution: Dynamic Positioning

The solution was dynamic positioning. We needed to:
1. Get Cursy's position on screen
2. Calculate the center point
3. Position the bubble above that point
4. Account for transforms and offsets

```javascript
function positionBubble(bubble, character) {
    const charRect = character.getBoundingClientRect();
    const roomRect = roomContainer.getBoundingClientRect();
    
    const charCenterX = charRect.left - roomRect.left + charRect.width / 2;
    const charCenterY = charRect.top - roomRect.top;
    
    bubble.style.left = (charCenterX + 10) + 'px'; // Slight right offset
    bubble.style.top = (charCenterY - 60) + 'px'; // Above character
}
```

It worked. Finally.

### The Z-Index Challenge

But then there was another problem: z-index.

We tried appending to different layers. Still behind.

### The Solution: Append to Container

The solution was to append bubbles to the room container, not the character layer. That way, they'd be above everything.

// Wrong:

roomContainer.appendChild(bubble);
